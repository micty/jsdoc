##对话框

>经典的网页对话框组件，内外皆用心雕琢。




- **支持普通与 12 方向气泡状对话框**
- **完善的焦点处理，自动焦点附加与回退**
- **支持 ARIA 标准**
- **面向未来：基于 HTML5 Dialog 的 API**
- **支持标准与模态对话框**
- **丰富且友好的编程接口**
- **能自适应内容尺寸**

>更多文档说明见 [http://aui.github.io/artDialog/doc/index.html](http://aui.github.io/artDialog/doc/index.html)
