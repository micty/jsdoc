
KISP 框架
==============================================================
--------------------------------------------------------------

###简介 

> KISP 是一个轻量级的 JavaScript 框架，采用 CMD 模式进行模块化的封装，
提供了一些模块和接口，*可以用于 Web 和轻应用的开发*。
